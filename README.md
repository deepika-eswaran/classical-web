# Bharatanatyam Learning App

An AI-powered Bharatanatyam learning platform designed to help beginners learn Bharatanatyam through interactive lessons, mudra exploration, audio guidance, and song analysis.

## Features

- Mudra learning with detailed descriptions
- Search and filter mudras
- Audio pronunciation support
- Song analysis and beat suggestion
- User-friendly learning interface
- Responsive design

## Tech Stack

### Frontend
- React.js
- HTML
- CSS
- JavaScript

### Backend
- Flask
- Python

## Installation

### Frontend

```bash
npm install
npm start
```

### Backend

```bash
pip install -r requirements.txt
python app.py
```

## Project Objective

To make Bharatanatyam learning more accessible through technology by combining traditional dance knowledge with modern AI-powered learning tools.
